# Digit Recognition Visualizer 🧠✍️

An interactive web-based project that demonstrates how a neural network “sees” and classifies handwritten digits.

Users can draw a digit on a 28×28 grid (similar to the MNIST dataset), and the model predicts the digit in real time. The prediction is visualized as a highlighted output among digits 0–9, helping users understand how neural networks interpret pixel data.

---

## 🚀 Features

- 28×28 interactive drawing grid
- Real-time digit prediction (0–9)
- Trained on the MNIST handwritten digits dataset
- Neural network inference directly in the browser using TensorFlow.js
- Clean and modular React-based frontend

---

## 🧠 How It Works

1. The user draws a digit on a 28×28 grid
2. The grid is converted into a 784-length input vector
3. A trained neural network processes the input
4. Output probabilities for digits 0–9 are generated
5. The predicted digit is highlighted visually
